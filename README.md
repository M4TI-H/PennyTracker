# PennyTracker - Personal Finance Management System

A full-stack financial application designed to help users track transactions and visualize their financial status. This project served as a core milestone in mastering Python-based backend development and integrating it with modern frontend frameworks.

## Technologies

* **Frontend:** Vue.js, TypeScript, Tailwind CSS
* **Backend:** FastAPI (Python)
* **Database:** MySQL
* **API Architecture:** RESTful API

## Key Features

* **Full-Stack Integration:** Engineered a seamless data flow between a reactive Vue.js interface and a high-performance Python backend.
* **Robust API Design:** Architected a RESTful API to handle secure transaction management.
* **Clean Architecture:** Developed a modular codebase focusing on scalability and maintainability, applying best practices for Python backend structure.
* **Real-time Data Visualization:** Created an intuitive dashboard using Tailwind CSS to provide users with immediate insights into their financial data.
* **UI/UX Design:** Created a Figma design beforehand to visualize project's functionality before development.

## Key Challenges & Learning Outcomes

* **First Python Backend:** Successfully navigated the transition to Python backend development, learning to manage asynchronous requests and database ORM.
* **Database Schema Design:** Designed a relational database structure in MySQL to ensure data integrity and efficient retrieval of complex financial records.
* **Bridging Ecosystems:** Gained hands-on experience in connecting disparate technologies (Vue.js & FastAPI) into one cohesive, functional ecosystem.
